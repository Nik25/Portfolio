# Niketan Hambarde - Portfolio

This is a static portfolio site built with HTML, CSS, JavaScript and Bootstrap.

## Add profile photo

Place your profile image at `assets/profile.jpg` relative to the project root. Recommended size: 600x600 or larger (square). The site will automatically use a placeholder image if the file is missing.

## Run locally

```powershell
cd 'C:/Desktop/portfolio new'
python -m http.server 5500 --bind 127.0.0.1
# open http://127.0.0.1:5500 in your browser
```

## Deploy

### GitHub Pages

1. Create a GitHub repository.
2. Commit these files to the repository root:
   - `index.html`
   - `style.css`
   - `script.js`
   - `Resume_NiketanHambarde (1).pdf`
   - `Nik photo new.jpeg`
   - `assets/`
3. Push to the `main` branch.
4. In the repo, go to **Settings > Pages**.
5. Set the source to `main` branch and folder `/ (root)`.
6. Save and visit the published URL.

### Netlify

- Drag and drop the project folder onto Netlify.
- Or connect GitHub and import the repository.
- No build command is required; publish directory is `/`.

### Local preview

```powershell
cd 'C:/Desktop/portfolio new'
python -m http.server 5500 --bind 127.0.0.1
```

Then open `http://127.0.0.1:5500`.
